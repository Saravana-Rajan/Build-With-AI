## Draft — "Explain your solution" (≤1000 chars)
> An AI decision-support platform for MPs that turns scattered, multilingual citizen
> demands (voice, text, photo, PDF — Tamil/Hindi/English) into a clear, defensible,
> ward/village-level action plan for Coimbatore. Multimodal Gemini normalises every
> channel into one structured record; near-duplicates are merged into real issues and
> coordinated campaigns flagged. Each need is routed to the responsible scheme —
> urban→AMRUT/PMAY-U, rural→Jal Jeevan/PMAY-G — and we compute the entitlement gap
> (eligible − already-covered) from real Census data: ₹463 crore owed-but-undelivered
> across Coimbatore. Two tracks: (A) unlock existing entitlements for ₹0 via
> auto-drafted department letters; (B) spend the MP's ₹5 cr only on true gaps via a
> budget optimiser. A "silent villages" layer surfaces high-need areas that never
> petition. Google Cloud native, deployable per constituency.

## Draft — "Technologies used" (≤1024 chars)
> Google Cloud (Vertex AI / Gemini 2.5-flash multimodal, BigQuery, Firestore, Cloud
> Run, Cloud Storage, Maps Platform), FastAPI (Python), React + TypeScript + Vite,

> Telegram Bot API, ReportLab (PDF letters), real data from data.gov.in (Census 2011
